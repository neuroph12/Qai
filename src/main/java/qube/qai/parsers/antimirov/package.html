Provides classes for representation of regular tree types and regular inequalities, including subtyping facility for regular tree types. For subtyping, also classes for the representation of sets, type pairs and names are provided.

Related Documentation

For a detailed description of the subtyping interface please see:
  • Stefan Hohenadel, Subtyping for Regular Tree Types: A JAVA-based Implementation. Master Thesis. pp.39-45.
